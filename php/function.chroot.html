chroot

(PHP 4 >= 4.0.5, PHP 5)

chroot — 改变根目录

说明

bool chroot ( string $directory )

将当前进程的根目录改变为 directory。

本函数仅在系统支持且运行于 CLI，CGI 或嵌入 SAPI 版本时才能正确工作。此外本函数还需要 root 权限。

参数

directory

新目录

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE.

注释

Note: 此函数未在 Windows 平台下实现。
