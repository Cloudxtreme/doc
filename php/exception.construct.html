Exception::__construct

(PHP 5 >= 5.1.0)

Exception::__construct — 异常构造函数

说明

public Exception::__construct() ([ string $message = "" [, int $code = 0 [, Exception $previous = NULL ]]] )

异常构造函数。

参数

message

抛出的异常消息内容。

code

异常代码。

previous

异常链中的前一个异常。
