shuffle

(PHP 4, PHP 5)

shuffle — 将数组打乱

说明

bool shuffle ( array &$array )

本函数打乱（随机排列单元的顺序）一个数组。

Note: 此函数为 array 中的元素赋与新的键名。这将删除原有的键名，而不是仅仅将键名重新排序。

Example #1 shuffle() 例子

<?php
$numbers = range(1,20);
srand((float)microtime()*1000000);
shuffle($numbers);
foreach ($numbers as $number) {
    echo "$number ";
}
?>

Note: 自 PHP 4.2.0 起，不再需要用 srand() 或 mt_srand() 给随机数发生器播种 ，因为现在是由系统自动完成的。

参见 arsort()，asort()，ksort()，rsort()，sort() 和 usort()。
