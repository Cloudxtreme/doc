安装

必须使用选项 --with-oracle[=DIR] 编译 PHP，DIR 的默认值是环境变量 ORACLE_HOME。

如果使用 » Oracle Instant Client，需要在编译 PHP 时加上 ">--with-oci8=instantclient,/path/to/oic/lib 选项。注意对 Oracle Instant Client 的支持最开始出现在版本 4.3.11 和 5.0.4。

Note:

注意在 OCI8 版本 1.1 之前必须用 --with-oci8-instant-client 来配置 PHP 和 Oracle Instant Client。
