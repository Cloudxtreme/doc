iterator_apply

(PHP 5 >= 5.1.0)

iterator_apply — 为迭代器中每个元素调用一个用户自定义函数

说明

ReturnType iterator_apply ( Traversable $iterator , callback $function [, array $args ] )

循环迭代每个元素时调用某一回调函数。

参数

iterator

需要循环迭代的类对象。

function

迭代到每个元素时的调用的回调函数。

args

传递到回调函数的参数。

返回值

返回已迭代的元素个数。

参见

  • array_walk() - 对数组中的每个成员应用用户函数
