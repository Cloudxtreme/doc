rewind

(PHP 4, PHP 5)

rewind — 倒回文件指针的位置

说明

bool rewind ( resource $handle )

将 handle 的文件位置指针设为文件流的开头。

成功时返回 TRUE， 或者在失败时返回 FALSE.

文件指针必须合法，并且指向由 fopen() 成功打开的文件。

Note:

如果将文件以附加（"a" 或者 "a+"）模式打开，写入文件的任何数据总是会被附加在后面，不管文件指针的位置。

参见 fseek() 和 ftell()。
