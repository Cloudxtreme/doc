pi

(PHP 4, PHP 5)

pi — 得到圆周率值

说明

float pi ( void )

返回圆周率的近似值。返回值的 float 精度是由 php.ini 中的 precision 指令确定。默认值是 14。你也可以使用 M_PI 常量，该常量产生与 pi() 完全相同的结果。

Example #1 pi()

<?php
echo pi(); // 3.1415926535898
echo M_PI; // 3.1415926535898
?>
