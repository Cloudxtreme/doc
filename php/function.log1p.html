log1p

(PHP 4 >= 4.1.0, PHP 5)

log1p — 返回 log(1 + number)，甚至当 number 的值接近零也能计算出准确结果

说明

float log1p ( float $number )
Warning

此函数是实验性的。 此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本中未通知就被修改。使用本扩展风险自担 。

log1p() 返回 log(1 + number)，甚至当 number 的值接近零也能计算出准确结果

Note: 此函数未在 Windows 平台下实现。

参见 expm1() 和 log()。
