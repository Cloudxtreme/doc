w32api_init_dtype

(PHP 4 >= 4.2.0)

w32api_init_dtype — 创建了一个数据类型的实例，并且将函数传入的值填入其中

描述

resource w32api_init_dtype ( string $typename , mixed $value [, mixed $... ] )
Warning

此函数是实验性的。 此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本中未通知就被修改。使用本扩展风险自担 。

这个函数创建了名为 typename 的数据类型的实例，并且填入了这种数据类型的数据。这种 typename 参数大小写敏感。你给出的值应该和你定义的 w32api_deftype() 类型顺序一致。返回资源的类型则为 dynaparm。
