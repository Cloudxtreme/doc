imagepsfreefont

(PHP 4, PHP 5)

imagepsfreefont — 释放一个 PostScript Type 1 字体所占用的内存

说明

bool imagepsfreefont ( resource $fontindex )

imagepsfreefont() 释放一个 PostScript Type 1 字体所占用的内存。

Note: 此函数仅在 PHP 编译时指定了 --with-t1lib[=DIR] 时可用。

参见 imagepsloadfont()。
