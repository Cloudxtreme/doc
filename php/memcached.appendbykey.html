Memcached::appendByKey

(PECL memcached >= 0.1.0)

Memcached::appendByKey — 向指定服务器上已存在元素后追加数据

说明

public bool Memcached::appendByKey ( string $server_key , string $key , string $value )

除了可以使用server_key自由的将key映射到指定服务器外， Memcached::appendByKey()在功能上等同于 Memcached::append()。 (译注: 关于*ByKey系列方法及$server_key的工作原理请参照addByKey方法文档)

参数

server_key

此键名标识存储值的服务器。

key

用于存储值的键名。

value

要追加的字符串。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE. 如果key不存在， Memcached::getResultCode()将返回 Memcached::RES_NOTSTORED。

参见

  • Memcached::append() - 向已存在元素后追加数据
  • Memcached::prepend() - 向一个已存在的元素前面追加数据
