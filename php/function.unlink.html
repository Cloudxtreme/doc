unlink

(PHP 4, PHP 5)

unlink — 删除文件

说明

bool unlink ( string $filename )

删除 filename。和 Unix C 的 unlink() 函数相似。成功时返回 TRUE， 或者在失败时返回 FALSE.

Note: 自 PHP 5.0.0 起 unlink() 也可以用于某些 URL 封装协议。参考Supported Protocols and Wrappers 中的列表看哪些封装协议支持 unlink()。

Note: 在 PHP 5.0.0 中增加了 对上下文(Context)的支持。 有关 上下文(Context) 的说明参见 Stream 函数。

参见目录删除函数 rmdir()。
