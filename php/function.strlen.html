strlen

(PHP 4, PHP 5)

strlen — 获取字符串长度

说明

int strlen ( string $string )

返回给定的字符串 string 的长度。

参数

string

需要计算长度的字符串。

返回值

成功则返回字符串 string 的长度；如果 string 为空，则返回 0。

范例

Example #1 strlen() 范例

<?php
$str = 'abcdef';
echo strlen($str); // 6

$str = ' ab cd ';
echo strlen($str); // 7
?>

参见

  • count() - 计算数组中的单元数目或对象中的属性个数
  • mb_strlen() - Get string length
