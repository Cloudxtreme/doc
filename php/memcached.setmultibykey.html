Memcached::setMultiByKey

(PECL memcached >= 0.1.0)

Memcached::setMultiByKey — Store multiple items on a specific server

说明

public bool Memcached::setMultiByKey ( string $server_key , array $items [, int $expiration ] )

除了可以使用server_key自由的将key映射到指定服务器外， Memcached::setMultiByKey()在功能上等同于 Memcached::setMulti()。 (译注: 关于*ByKey系列方法及$server_key的工作原理请参照addByKey方法文档)。

参数

server_key

此键名标识存储值的服务器。

items

存放在服务器上的键/值对数组。

expiration

到期时间，默认为 0。 更多 信息请参见到期时间。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE. 如需要则使用 Memcached::getResultCode()。

参见

  • Memcached::setMulti() - 存储多个元素
  • Memcached::set() - 存储一个元素
